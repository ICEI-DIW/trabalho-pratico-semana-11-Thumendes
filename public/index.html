Sobre

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro, com o objetivo de apresentar uma lista de pontos turísticos. A aplicação utiliza a API do Unsplash para buscar imagens e informações sobre os pontos turísticos utilizando IA.
